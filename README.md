# Projeto Python Pro - Gerador de Senhas com Tkinter

Este projeto foi desenvolvido como parte da tarefa prática para tutores da Kodland. O objetivo é demonstrar a aplicação de interfaces gráficas (GUI) utilizando a biblioteca nativa **Tkinter**.

# Funcionalidades
- Geração Personalizada: O utilizador define o comprimento da senha.
- Segurança: Utiliza uma combinação de letras (maiúsculas e minúsculas), números e caracteres especiais.
- Tratamento de Erros: Validação para garantir que apenas números sejam inseridos no campo de tamanho.
- Interface Intuitiva: Janela limpa e funcional para facilitar a experiência do utilizador.

# Critérios Atendidos
- [x] Utilização da biblioteca Tkinter.
- [x] Uso de funções integradas da biblioteca.
- [x] Inclusão de arquivo `requirements.txt`.
- [x] Documentação via `README.md`.

# Como executar
1. Certifique-se de ter o Python instalado.
2. Clone este repositório.
3. Execute o ficheiro principal:
   ``python main.py``
